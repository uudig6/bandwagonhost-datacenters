## 二、为什么搬瓦工机房这么多：线路类型的底层逻辑

机房数量多，本质上是线路类型多。搬瓦工最大的卖点，就是把中国电信、联通、移动三网的优质回程线路分别接到了不同机房，让用户按预算和需求"对号入座"。把这层逻辑理顺了，所有机房你都能一眼看懂。

**1. 普通直连线路（KVM 系列主打）**
走的是国际公网，到国内三网都是普通 BGP，没有专门优化。优点是便宜，洛杉矶 DC2 / DC4 / DC8、弗里蒙特、纽约、新泽西、加拿大、荷兰 EUNL_3、悉尼、迪拜都属于这一类。适合预算紧、对延迟不敏感的场景，比如挂梯子后台、海外爬虫、给国外用户访问的小站。

**2. CN2 GT 线路**
电信第二代承载网，但属于"半优化的 GT"，回程高峰期会绕路。搬瓦工里 DC3 就是典型 CN2 GT 机房。性价比中等，比普通直连强一点，但又够不到 GIA 的水准。

**3. CN2 GIA / CN2 GIA-E 线路（搬瓦工的核心招牌）**
这是电信花钱维护的精品网，回程全程走 AS4809，国内三网高峰期也不绕路、不掉速。其中 DC9 是 CN2 GIA，DC6 是 CN2 GIA-E（E 表示 E-commerce，带宽更大、给电商场景准备的）。香港、东京、大阪、新加坡、加拿大温哥华 CABC_6、荷兰 EUNL_9 也都接入了 CN2 GIA。这一档是搬瓦工真正"贵有贵的道理"的地方，也是大多数人买搬瓦工的核心动机。

**4. CMIN2 / CUP 线路**
CMIN2 是移动的精品回程，CUP 是联通的 AS9929 / AS10099 高级回程。搬瓦工部分高端套餐会在 CN2 GIA 之外再叠加这些线路做"三网优化"，比如荷兰 EUNL_9 就是联通 AS10099 + 电信 CN2 GIA 的组合，对联通用户特别友好。

一句话总结：**普通直连 = 够用，CN2 GT = 略好，CN2 GIA / GIA-E = 主推，叠加 CMIN2/CUP = 顶配。**
